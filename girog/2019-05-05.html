엑셀 공부

동적범위

  • 필요성: 항목이 추가될 때 마다 합계를 내는 등 계산이 필요할 경우 범위 참조를 명시적으로 작성해두면 범위를 일일이 다시 바꿔주어야 해서 불편함. 이를 위해서 자료의 변화에 따라 자동으로 범위가 변할 수 있도록 하는 방법이 필요함.

  • 엑셀의 '이름 관리자'의 참조 대상에 OFFSET() 함수와 COUNTA() 함수를 통해 작성하면 데이터 변화에 따라 수식들이 자동으로 범위를 잡아줌.

  • OFFSET(기준이 되는 지점, 행 이동할 값, 열 이동할 값, 행 범위, 열 범위)

  • COUNTA(범위)

  • EX) C1은 목록의 이름이며 C 한 줄에 있는 모든 목록을 동적범위로 잡는다고 가정

    -> OFFSET($C$1,0,0, COUNTA($C:$C)-1,1)

- 목록 이름인 C1을 절대참조로 기준으로 잡는다.
- C1에서 기준을 이동하지 않을 것이므로 0, 0
- COUNTA($C:$C) C 한 줄을 절대참조하여 비어 있지 않은 셀의 개수를 행 범위로 잡아준다.
- 목록 제목은 범위에 포함하지 말아야 하므로 -1
- 한 줄만 범위로 지정할 것이므로 마지막 인자는 1
